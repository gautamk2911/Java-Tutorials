## ğŸ§  What are Generics?

- **Generics** allow you to create **classes**, **interfaces**, and **methods** with a **placeholder** for the data type.
- It helps in:
  - Code **reusability**
  - **Type safety** (no `ClassCastException` at runtime)
  - Reducing **code duplication**

```java
// Without Generics
List list = new ArrayList();
list.add("Java");
String s = (String) list.get(0); // Manual type-casting

// With Generics
List<String> list = new ArrayList<>();
list.add("Java");
String s = list.get(0); // No casting needed
```

### âœ… Bounded Types (`<T extends ClassName>`)

You can restrict generics to **only certain types**.

```java
public class NumberBox<T extends Number> {
    private T num;

    public void setNum(T num) {
        this.num = num;
    }

    public T getNum() {
        return num;
    }
}

public class TestNumberBox {
    public static void main(String[] args) {
        NumberBox<Integer> intBox = new NumberBox<>();
        intBox.setNum(50);

        NumberBox<Double> doubleBox = new NumberBox<>();
        doubleBox.setNum(20.5);

        // NumberBox<String> stringBox = new NumberBox<>(); // âŒ Error
    }
}
```

âœ… `T` must be a subclass of `Number`.

## ğŸ”¹ Wildcards, Diamond Operator, Real-use Examples

### âœ… Wildcards `<?>`

Wildcards are used when you want **flexibility**.

```java
import java.util.List;

public class WildcardExample {
    public static void printList(List<?> list) {
        for (Object obj : list) {
            System.out.println(obj);
        }
    }
}
```
âœ… Here, `?` can accept any type â€” `List<Integer>`, `List<String>`, etc.

---

### âœ… Upper Bounded Wildcards (`<? extends Class>`)

```java
public static void sumNumbers(List<? extends Number> list) {
    double sum = 0;
    for (Number num : list) {
        sum += num.doubleValue();
    }
    System.out.println(\"Sum: \" + sum);
}
```

âœ… Accepts list of any type that extends `Number` (like `Integer`, `Float`, `Double`).

---

### âœ… Lower Bounded Wildcards (`<? super Class>`)

```java
public static void addNumbers(List<? super Integer> list) {
    list.add(10);
    list.add(20);
}
```

âœ… Accepts list of `Integer`, `Number`, or `Object` types.
